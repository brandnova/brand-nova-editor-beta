Configuration Options

The Valiux Editor supports extensive customization through its configurable toolbar system. You can create editors ranging from minimal comment boxes to full-featured content editors.

Editor Modes

The editor operates in two main modes:

  • Full Editor: Shows all available tools organized in logical groups
  • Partial Editor: Shows only the tools specified in your custom configuration

Toolbar Groups

Tools are organized into the following groups:

  • formatting: bold, italic, underline, strikethrough
  • headings: heading-one, heading-two, heading-three, paragraph
  • alignment: left, center, right, justify
  • blocks: block-quote, bulleted-list, numbered-list
  • actions: undo, redo
  • fullscreen: fullscreen

Common Configuration Examples

Basic Writing Editor

Perfect for simple content creation with essential formatting tools

{
  formatting: ['bold', 'italic'],
  headings: ['heading-one', 'heading-two', 'paragraph'],
  actions: ['undo', 'redo']
}

Comment Editor

Minimal editor for user comments with basic formatting

{
  formatting: ['bold', 'italic', 'strikethrough']
}

Blog Post Editor

Comprehensive editor for content creation with full formatting options

{
  formatting: ['bold', 'italic', 'underline'],
  headings: ['heading-one', 'heading-two', 'heading-three', 'paragraph'],
  blocks: ['block-quote', 'bulleted-list', 'numbered-list'],
  actions: ['undo', 'redo'],
  fullscreen: ['fullscreen']
}

Social Media Editor

Simple formatting for social media posts

{
  formatting: ['bold', 'italic', 'strikethrough'],
  blocks: ['bulleted-list']
}

Keyboard Shortcuts

The editor supports standard keyboard shortcuts that are displayed in tooltips:

  • Ctrl/Cmd + B: Bold
  • Ctrl/Cmd + I: Italic
  • Ctrl/Cmd + U: Underline
  • Ctrl/Cmd + Shift + S: Strikethrough
  • Ctrl/Cmd + Z: Undo
  • Ctrl/Cmd + Y: Redo
  • F11: Fullscreen

Usage in Your Django Templates

After building the standalone version, include it in your Django templates like this:

{% load static %}
<link rel="stylesheet" href="{% static 'js/brand-nova-editor.css' %}">

<div id="my-editor" 
     data-brand-nova-editor
     data-full-editor="false"
     data-toolbar-config='{"formatting": ["bold", "italic"], "actions": ["undo", "redo"]}'
     data-theme="light"
     data-placeholder="Start writing..."></div>

<script src="{% static 'js/brand-nova-editor.js' %}"></script>
